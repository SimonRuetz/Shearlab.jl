Shearlab.jl

Shearlab is a Julia Library with toolbox for two- and threedimensional data processing using the Shearlet system as basis functions which generates an sparse representation of cartoon-like functions. It is based in the Matlab Library Shearlab3D, developed by the Applied Functional Analysis Research Group in the Technical University of Berlin lead by Professor Gita Kutyniok, for further information of the Matlab Implementation you can visit the link Shearlab3D.

Shearlab.jl is maintained by Héctor Andrade Loarca.
